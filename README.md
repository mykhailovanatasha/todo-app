# AI-планер дня

Mobile-first todo-застосунок: диктуєш або пишеш усе, що в голові, — AI перетворює хаос на структуровані задачі та план на сьогодні.

**Поточний стан:** каркас із 3 екранів без AI-парсингу.

## Екрани

- **Capture** (`/`) — велике поле «Що в голові?» + кнопка мікрофона (диктування через Web Speech API, uk-UA). Записи зберігаються як «сирі» нотатки.
- **Inbox** (`/inbox`) — список розпарсених задач із пріоритетом, часом і дедлайном (поки порожній).
- **Today** (`/today`) — чекліст задач на сьогодні (поки порожній).

## Стек

- Next.js 15 (App Router) + React 19 + TypeScript
- Tailwind CSS 4
- Стан: React state + `localStorage` (`lib/store.ts`), без бекенду

## Запуск

```bash
npm install
npm run dev
```

Відкрий http://localhost:3000 (найкраще виглядає в мобільному режимі DevTools або на телефоні).
